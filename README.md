### Sistema de Análise de Irrigação Agrícola

Repositório reservado para a resolução do desafio proposto como etapa do módulo de Algoritmos Supervisionados, da formação de Machine Learning da plataforma Rocketseat. 

O objetivo é desenvolver um sistema completo de análise de dados de irrigação, com a construção de um pipeline capaz de carregar dados, realizar análises exploratórias, construir um modelo preditivo com algoritmo de regressão linear simples e avaliar resultados para otimizar o uso de recursos hídricos de uma fazenda.

O que foi solicitado:
- Carga dos dados
- Visualização da estrutura das variáveis
- Cálculo das estatísticas descritivas
- Criação de gráfico de dispersão
- Análise de correlação entre atributos
- Divisão dos dados para treino e teste
- Treino do modelo
- Impressão da equação da reta
- Avaliação do modelo a partir das métricas de desempenho MSE e MAE
- Visualização dos resultados reais e preditos no gráfico
- Cálculo e análise de resíduos do modelo
- Verificação da normalidade dos resíduos com gráficos e testes estatísticos
- Predições a partir do modelo

Bibliotecas do Python utilizadas:
- Pandas
- Matplotlib
- Seaborn
- Scipy
- Sklearn
- Statsmodels
